ประวัติการทำงานที่ผ่านมา

บริษัท บุ๊คเอนจิ้นดอทคอม (ประเทศไทย) จำกัด

รูปแบบการจ้าง : Full-Time พนักงานประจำ
ระยะเวลาการทำงาน : 03/12/2567 - ปัจจุบัน ( Time )
ที่ตั้ง : 91 หมู่ที่ 1 ต.บางกรวย อ.บางกรวย จ.นนทบุรี 11130

บริษัท บุ๊คเอนจิ้นดอทคอม (ประเทศไทย) จำกัด เป็นบริษัทที่จัดทำโปรแกรมเว็บเพจและเครือข่ายตามวัตถุประสงค์ของผู้ใช้ อยู่ในเครือของโรงแรม ทำเกี่ยวกับระบบจองห้องพักออนไลน์ และธุรกิจต่าง ๆ เกี่ยวกับโรงแรม

งานที่ได้รับมอบหมาย
  1. ระบบขอออกนอกสถานที่สำหรับพนักงาน
  2. ระบบยืมของสำหรับพนักงานและลูกค้า
  3. ระบบขอตำแหน่งพนักงานและผู้สมัคร
  4. ระบบลา (เพิ่มการตรวจสอบการลายาว และเงื่อนไขการลาประเภทต่าง ๆ )
  5. api ข้อมูล และ export files 
  6. ปรับปรุง permissions

บริษัท บิซโพเทนเชียล จำกัด

รูปแบบการจ้าง : Full-Time พนักงานประจำ
ระยะเวลาการทำงาน : 02/05/2565 - 24/05/2567 ( Time )
ที่ตั้ง : 807/1 ถ. พระรามที่ 3 แขวงบางโพงพาง เขตยานนาวา กรุงเทพมหานคร 10120

บริษัท บิซโพเทนเชียล จำกัด เป็นบริษัทพัฒนา Software ด้าน Web Based Application พัฒนาระบบตามความต้องการของผู้ใช้งาน โดยให้บริการทั้งราชการ รัฐวิสาหกิจ สถาบันการศึกษา และเอกชน ปัจจุบันมีโครงการที่จัดทำแล้วกว่า 800 โครงการ ก่อตั้งเมื่อวันที่ 16 พฤษภาคม 2543 โดยคุณวิทูร หวังสงวนกิจ (ปัจจุบันดำรงตำแหน่ง กรรมการผู้จัดการ)

งานที่ได้รับมอบหมาย
  1. ระบบยื่นคำร้องทางอิเล็กทรอนิกส์ E-Filing กรมบังคับคดี
  2. ระบบงานบูรณาการฐานข้อมูล กรมบังคับคดี
  3. ระบบประเมิน KPI และ Competency กรมประชาสัมพันธ์
  4. ระบบนำเข้าและแสดงสลิปเงินเดือน / ระบบจัดการผู้ใช้งาน กรมส่งเสริมอุตสาหกรรม
  5. อื่น ๆ

ระบบยื่นคำร้องทางอิเล็กทรอนิกส์ E-Filing กรมบังคับคดี หน้าที่ที่ได้รับมอบหมายคือ การสร้าง แก้ไข ตรวจสอบและปรับเปลี่ยน หน้าจอแสดงผล ฟอร์มบันทึกข้อมูล ของคำร้องต่าง ๆ ตามประเภทของคำร้องหรือคดีที่ตาม PM หรือ BA แจ้ง

ระบบงานบูรณาการฐานข้อมูล กรมบังคับคดี หน้าที่ที่ได้รับมอบหมายคือ ระบบจัดการผู้ใช้งานและการให้สิทธิ์เข้าใช้งาน API ของผู้ใช้งานแต่ละหน่วยงาน, การดึงข้อมูลจากระบบต่าง ๆ มารวมกันเพื่อจัดการข้อมูลให้เป็นระเบียบและนำไปใช้งานต่อ

ระบบประเมิน KPI และ Competency กรมประชาสัมพันธ์ หน้าที่ที่ได้รับมอบหมายคือ ทำระบบแบบประเมินพนักงานราชการ ลูกจ้างชั่วคราว และพนักงานราชการ โดยแบบประเมินจะประกอบไปด้วยการประเมิน ดัชนีชี้วัดผลงานหรือความสำเร็จของงาน (KPI) และสมรรถนะการแสดงออกในเชิงพฤติกรรม เพื่อทำงานให้ประสบผลสำเร็จ (Competency)

*ทำระบบแบบประเมินคล้ายกันที่
  1. กองทุนพัฒนาผู้ประกอบการเทคโนโลยีและนวัตกรรม (TEDFUND)
  2. สำนักงานบริหารกองทุนเพื่อส่งเสริมการอนุรักษ์พลังงาน (ENCONFUND)

ระบบนำเข้าและแสดงสลิปเงินเดือน / ระบบจัดการผู้ใช้งาน กรมส่งเสริมอุตสาหกรรม

  • ระบบนำเข้าและแสดงสลิปเงินเดือน หน้าที่ที่ได้รับมอบหมายคือ ทำระบบโดยมีกระบวนการคืออ่านไฟล์สลิปเงินเดือนที่ทางเจ้าหน้าที่นำเข้าสู่ระบบ และทำการ insert เข้าสู่ระบบ และจากนั้น นำข้อมูลไปแสดงผลที่หน้าแสดงสลิปเงินเดือนของผู้ใช้งานแต่ละคน โดยปรับแก้การแสดงผลตามที่ฝ่ายออกแบบได้เตรียมไว้แล้ว
  • ระบบจัดการผู้ใช้งาน หน้าที่ที่ได้รับมอบหมายคือ ทำแบบฟอร์มสมัครสมาชิก รวมถึงขั้นตอนการอนุมัติต่าง ๆ รวมไปจนถึงจัดการสิทธิ์เข้าถึงแต่ละระบบ

อื่นๆ

  • มีงานย่อย ๆ อีกหลายโครงการ เนื่องจากไม่ได้เป็นโครงการหลัก แต่จะเป็นการไปช่วยเหลืองานโครงการนั้น ๆ เช่น ช่วยแก้ไขบัค ปรับแก้ไขการทำงานของระบบบางส่วน เป็นต้น

บริษัท ซิตี้วาไรตี้ คอร์เปอเรชั่น จำกัด

รูปแบบการจ้าง : Contract รายวัน (เดือนต่อเดือน)
ระยะเวลาการทำงาน : 18/03/2565 ถึง 31/05/2565 ( Text )
ที่ตั้ง : 24 ซ.4 หมู่บ้านทานตะวัน ต.คอหงส์ อ.หาดใหญ่ จ.สงขลา 90110

บริษัท ซิตี้วาไรตี้ คอร์เปอเรชั่น จำกัด เป็นบริษัทที่รับออกแบบและพัฒนา Web site รวมถึง Mobile Application ให้กับหน่วยงานทั้งภาครัฐและเอกชน เปิดดำเนินงานมาแล้วมากกว่า 10 ปี

งานที่ได้รับมอบหมาย

แก้ไขการแสดงผลเว็บไซต์และเพิ่มระบบจัดการหลังบ้าน เช่น ระบบเอกสารสำหรับประชาชน โดยเจ้าหน้าที่ของหน่วยงาน (ในที่นี้คือ อบต) เพิ่มข้อมูลจากหลังบ้านและนำมาแสงผลบนหน้าเว็บไซต์ หรือ ระบบข่าวประชาสัมพันธ์ โดยเพิ่มภาพสำหรับการประชาสัมพันธ์จากเจ้าหน้าที่ ที่หลังบ้านเพื่อแสดงประกาศที่หน้าเว็บเมื่อเปิดเข้าหน้าเว็บ เพื่อเป็นการประชาสัมพันธ์ข่าวสารในช่วงงาน หรือเทศกาศต่าง เป็นต้น

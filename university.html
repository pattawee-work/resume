[ 0214493 ] โครงงานวิจัยทางวิทยาการคอมพิวเตอร์

   โครงงานวิจัยเรื่อง "แอปพลิเคชันจำแนกสายพันธุ์ทุเรียนด้วยเทคนิคการเรียนรู้ด้วยเครื่อง ( Application for Durian Variety Classification using Machine Learning Technique )" เป็นการทำโครงงานวิจัยด้วยวิธีการสร้างโมเดล Machine learning และพัฒนาร่วมกับการเขียนแอปพลิเคชันด้วย Flutter เพื่อสร้างเป็นแอปที่ช่วยอำนวยความสะดวกในการจำแนกสายพันธุ์ทุเรียน

[ 0214495 ] การฝึกสหกิจศึกษา

   สหกิจศึกษาที่ บริษัท ซิตี้วาไรตี้ คอร์เปอเรชั่น จำกัด ตั้งอยู่ที่ ตำบลคอหงส์ อำเภอหาดใหญ่ สงขลา 90110 โดยเริ่มต้นการปฏิบัติงานสหกิจศึกษาวันที่ 29 พฤศจิกายน พ.ศ. 2564 สิ้นสุดการปฏิบัติสหกิจศึกษา วันที่ 18 มีนาคม พ.ศ. 2565 ทางบริษัทเป็น Software House จัดทำ website หรือออกแบบตามความต้องการของผู้ใช้งาน เครื่องมือหลักคือ Codeigniter (PHP Framework) โดยระหว่างการสหกิจศึกษา ได้มีการทำโปรเจคชื่อ แอปพลิเคชันระบบบันทึกการตรวจเมมโมรี่การ์ดกล้องติดรถ (Application to Record the Inspection of Car Camera Memory Card) เป็นแอปพลิเคชันเพื่อใช้สำหรับการตรวจกล้องติดรถ เพื่ออำนวยความสะดวกให้กับเจ้าหน้าที่ ที่ทำหน้าที่ตรวจสอบความพร้อมของกล้องทุกเดือน
